# AccountBook_26

这是一个基于Python开发的简易个人记账工具，旨在帮助用户快速记录、查询和统计个人收支情况。

## 功能介绍
- **收支记录**：支持记录收入/支出的金额、类别、日期和备注信息
- **数据查询**：可按日期范围、收支类别筛选查看记录
- **简单统计**：自动计算月度收支总和、各类别支出占比
- **dev-record分支**：完成记账功能代码开发
- main分支：完成项目基础文档搭建 

## 技术栈
- 开发语言：Python 3.x
- 数据存储：CSV文件（基础版）/ SQLite数据库（进阶版）
- 依赖包：pandas（用于数据处理与统计）

## 开发计划
- `dev-record`分支：开发收支记录核心功能
- `dev-query`分支：开发数据查询功能
- `dev-stat`分支：开发收支统计功能
